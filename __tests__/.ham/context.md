# __tests__ — Test suites

Inherits from: root .ham/context.md

## Conventions
- Test files mirror the source directory structure
- Each test file focuses on one module or component
- Use Jest + Testing Library + Playwright for testing
- Use descriptive test names that explain the expected behavior
- Prefer integration tests for API endpoints, unit tests for utilities
- Mock external dependencies, not internal modules
